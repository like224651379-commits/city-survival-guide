# Gitee Pages 部署指南

## 1. 注册 & 实名（2分钟）
- 打开 https://gitee.com
- 用邮箱注册，登录
- 点右上角头像 → 设置 → 实名认证（需要身份证+人脸识别）

## 2. 创建仓库（1分钟）
- 点右上角 + → 新建仓库
- 仓库名：`city-survival-guide`
- 选择：公开
- 勾选：使用 README 文件初始化（可选）

## 3. 导入代码（1分钟）
有两种方式：

### 方式A：从 GitHub 导入（推荐）
- 仓库页面 → 管理 → 仓库设置 → 同步 → 从 GitHub 导入
- 填入：`https://github.com/like224651379-commits/city-survival-guide`
- 点导入

### 方式B：本地上传
- 下载 GitHub 仓库 zip，解压
- 拖拽上传到 Gitee 仓库

## 4. 开启 Gitee Pages（1分钟）
- 进入仓库 → 服务 → Gitee Pages
- 部署分支：master 或 main
- 部署目录：/（根目录）
- 点启动

## 5. 完成
- 等待 2-5 分钟
- 获得域名：`https://like224651379.gitee.io/city-survival-guide`
- 这个域名国内访问 100% 稳定
